# 🎬 KEPELIS - Streaming Platform

Kepelis is an interactive web application designed to explore movie and TV show trends in real-time. Inspired by modern streaming platforms, it offers an immersive visual experience for cinema lovers.  

## 🚀 Key Features
* **Dynamic Carousel**: A stunning features banner using `Swiper.js` for smooth, high-quality transitions.
* **Cinematic UI**: A modern, dark-themed responsive design optimized for a premium viewing experience.
* **Content Discovery**: Real-time trend visualization powered by the TMDB API.
* **Component-Based Architecture**: Modular React structure for easy scalability and maintenance.

## 🛠️ Technologies Used
* **React.js**: Core library for the user interface.
* **Vite**: Fast build tool for optimized development.
* **Swiper.js**: Interactive carousel implementation.
* **TMDB API**: Real-time cinematic data fetching.
* **CSS3**: Custom styling to achieve a professional aesthetic.

## 📦 Installation & Local Usage

1. Clone the repository:
    git clone https://github.com/KeylaConde/kepelis.git

2. Install dependencies:
   npm install

3. Start the development server:
   npm run dev
